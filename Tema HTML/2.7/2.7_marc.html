ATRIBUTO "MULTIPLE"

El atributo multiple es un atributo Booleano de los elementos input. Especifica que al usuario le está permitido seleccionar más de un valor que representa un elemento input. El atributo "multiple" funciona con muchos tipos de input, como emails, subir archivos, etc.

< form>
Select images:
< input type="file" multiple> < input type="submit">
< / form>





DATALIST

El tag datalist especifica una lista de opciones predefinidas para un elemento input. Es usado para que el usuario obtenga una lista desplegable que se autocompleta mientras escribe. Si clicas dos veces sobre el input también puedes ver la lista.

< label>Choose your browser from the list:< /label>
< input list="browsers">
< datalist id="browsers">
< option value="Edge">
< option value="Firefox">
< option value="Chrome">
< /datalist>





ATRIBUTO AUTOFOCUS

El atributo autofocus es un atributo booleano. Simplemente especifica que cuando se carga la página, su elemento input estará resaltado (focused).

< form>
< label>First name< /label>
< input type="text" autofocus>
< label>Last name:< /label>
< input type="text">
< input type="submit">
< /form>
